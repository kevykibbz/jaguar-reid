# Project List

| Patterns |
| --- |
| By Grow Jungles and Las Oncas |
| Patterns is an AI-powered wildlife monitoring platform built to transform how we study and protect biodiversity. Designed specifically for camera trap imagery, Patterns identifies and tracks species—starting with jaguars—by combining cutting-edge computer vision with ecological metadata. It’s a collaboration between two conservation-driven organizations, Grow Jungles and Las Oncas, with deep roots in field research and rainforest protection. |
|  |
| Why Patterns Exists |
| Traditional wildlife monitoring is time-consuming, fragmented, and often inaccessible to smaller conservation teams. As biodiversity loss accelerates, we need faster, smarter, and more scalable ways to track species across landscapes. Patterns was born out of necessity in the field—to automate species identification, map animal movements, and unlock the full potential of the data already being collected in rainforests around the world. |
|  |
| Vision |
| To create a global platform that empowers local conservationists, scientists, and indigenous communities with real-time wildlife intelligence—bridging field observation with AI to protect species before it's too late. |
|  |
| Mission |
| To democratize wildlife monitoring through AI, enabling any team with a camera trap to: |
| Identify species automatically |
| Track individual animals over time |
| Visualize movement patterns with spatial and temporal precision |
| Share insights that inform real conservation decision |
